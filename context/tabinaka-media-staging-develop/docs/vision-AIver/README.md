## 🚀 はじめに

### 開発者向け

```bash
# リポジトリのクローン
git clone https://github.com/your-org/gappy-v2.git

# 依存関係のインストール
npm install

# 環境変数の設定
cp .env.example .env.local
# .env.localに以下を設定:
# OPENAI_API_KEY=
# GOOGLE_PLACES_API_KEY=
# NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
# SUPABASE_URL=
# SUPABASE_ANON_KEY=

# 開発サーバーの起動
npm run dev
```
