## マイクロエクスペリエンスとは？

### 定義

**日常の中にある、小さいけれど魅力的な体験**

- 時間: 5分〜60分程度
- 価格: ¥0〜¥2,000程度
- 予約: 不要
- 場所: どこでもできる
- 制約: なし

### 特徴

1. **いつでもできる** - 予約不要、思い立ったら即実行
2. **価格が明確** - ¥300のファミチキ、¥150の自販機飲料
3. **ハードルが低い** - 誰でも気軽に試せる
4. **文化体験** - 日本の日常文化に触れる
5. **組み合わせ可能** - 複数の体験を1日に詰め込める

├─ データ販売（匿名化された体験データ）

利点:
✅ 無限のコンテンツ

### 検索機能拡張

```typescript
// マイクロエクスペリエンスに特化したフィルター
interface MicroExperienceFilters {
  priceRange: {
    min: number;
    max: number; // 通常は¥0-¥2000
  };
  duration: {
    max: number; // 60分以内
  };
  bookingRequired: boolean; // falseのみ
  availableNow: boolean; // 今すぐできる
  distance: {
    from: Location;
    maxDistance: number; // km
  };
  difficulty: 'beginner' | 'intermediate' | 'advanced';
  chainStore: boolean; // チェーン店のみ
}

// 検索クエリ例
const searchMicroExperiences = async (filters: MicroExperienceFilters) => {
  return await supabase
    .from('activities')
    .select('*')
    .eq('experience_type', 'micro-experience')
    .eq('booking_required', false)
    .lte('price', filters.priceRange.max)
    .lte('duration_minutes', filters.duration.max)
    .eq('approval_status', 'approved');
};
```
