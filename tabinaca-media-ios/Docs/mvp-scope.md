# MVPスコープ（Phase 1 定義）

## 対象範囲（MVPに含める）
- Home
- Experiences（一覧・詳細・Like）
- Chat（基本会話 + Placeカード）
- Quiz（基本フロー + 結果表示）
- Articles（一覧・詳細）
- 認証（Supabase OAuth / セッション管理）
- 位置情報（取得・許可フロー）

## MVPでは後回し（Phase 2+）
- QRコード/トラッキング/レビュー/事業者向け画面
- チャット共有リンクの公開ビュー（閲覧専用ページ）
- AI生成保存カードの詳細な再現
- 高度なアナリティクス連携

## 補足
- Web APIの利用可否が確定しない場合は、MVPではモック/固定データで進行。
- QR系の機能はビジネス用途のため、iOSはWebViewでの代替を検討。
